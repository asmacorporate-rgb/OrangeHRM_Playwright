**Playwright TypeScript Project Setup**

- This repository contains a Playwright testing framework with TypeScript, set up for collaborative work and clean coding practices.

- Create Github Account using AGT Email
- Setup Github Desktop with our Github Account
- Create a Repository
- Create a Branch in that Repository
- Create a Playwright Typescript Project in that specific Branch of the Repository
- Add Git Ignore
- Add TSConfig, Prettier, Eslint to the Project
- Make sure Eslint and Prettier are working
- Push to your Branch
